# Instrucciones


## Para instalar pipenv en linux (wsl):

En el directorio base ubuntu de tu computador corre:

1. sudo apt update
2. sudo apt install pipenv


## Para iniciarlo en wsl:
En la carpeta del proyecto:

La primera vez correr:
    1. pipenv install django

Si django ya está instalado:
    1. pipenv shell para ingresar al ambiente de producción 
    2. python manage.py runserver
